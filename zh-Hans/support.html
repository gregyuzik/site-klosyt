常见问题

📸

拍出完美的照片

为获得最佳效果，将你的衣物悬挂在纯色对比背景前。确保光线明亮均匀，帮助我们的AI准确去除背景并捕捉真实颜色。

📥

如何导入

通过拍照、从相册导入或粘贴服装店URL来添加物品。你还可以从任何应用直接分享——在Safari中浏览并点击分享即可发送到Klosyt。Klosyt会自动去除背景，让你的衣橱看起来整洁美观。

👗

创建搭配

前往搭配标签页混搭你的上衣、下装和鞋子。按穿着顺序拖动和整理层次，保存喜爱的组合以备后用。

✨

使用AI建议

不知道穿什么？点击AI造型师按钮！我们的设备端智能分析实时天气和你的真实衣橱，为每种活动推荐完美搭配。

☁️

备份与恢复

你的Klosyt数据非常珍贵。在应用设置中启用iCloud同步，安全备份整个衣橱。获得新设备后，使用相同的Apple ID登录，你的衣橱将无缝恢复。

📱

多平台

你的数字衣橱随你而行。Klosyt是iPhone、iPad、Mac、Apple TV和Vision Pro上的原生应用——你的衣橱始终触手可及。

🏷️

管理你的物品

点击任何物品查看详情。编辑类别、季节、颜色、材质和图案。直接从详情页重新拍照。删除的物品会先移至垃圾箱，方便恢复。

📅

日历与计划

使用日历标签为即将到来的日子或活动规划搭配。将保存的搭配分配到特定日期，追踪穿着记录，并与Apple日历同步。

🔗

分享与导出

将喜爱的搭配分享为精美的Pantone风格卡片。使用iOS分享扩展从Safari、Instagram或任何应用直接导入图片或URL。

🔍

筛选与排序

Klosyt的GelCap筛选系统让你即时缩小衣橱范围。按类别、季节、颜色、材质或图案筛选。按名称搜索，在网格和衣橱视图间切换。

🎨

主题与壁纸

让Klosyt成为你自己的！从精美的壁纸和主题中选择来自定义应用的外观和风格——完全免费，无需订阅。你选择的主题会在所有Apple设备间同步。

📦

Object Capture（3D扫描）

在支持LiDAR的iPhone Pro机型上，创建令人惊叹的衣物3D扫描。围绕物品走动以捕捉每个角度。3D模型连同标准照片一起保存。

故障排除

iCloud同步不工作
确保在设置 → Apple ID → iCloud中为Klosyt启用了iCloud。确保在所有设备上使用相同的Apple ID登录。尝试在Klosyt设置中关闭再开启iCloud同步。
照片无法加载
检查Klosyt是否在设置 → Klosyt → 照片中有访问照片库的权限。如果图片显示空白，尝试切换iCloud同步。
AI建议似乎不准确
确保你的物品有准确的季节标签和类别。AI使用这些元数据和天气数据。衣橱中物品越多 = 建议越好。
订阅未被识别
前往Klosyt设置并点击"恢复购买"。确保使用购买时的Apple ID登录。
背景去除不干净
为获得最佳效果，在纯色对比背景上拍摄衣物，确保光线均匀。避免复杂的背景或阴影。
天气或位置未更新
确保在设置 → 隐私与安全 → 定位服务中为Klosyt启用了定位服务。选择"使用应用期间"或"始终"以允许天气更新。
物品显示模糊或低质量
图片可能仍在从iCloud下载中。确保网络连接稳定并等待同步完成。应用内拍摄的照片以全分辨率存储。
如何删除一个物品？
点击一个物品 → 详情视图 → 删除按钮。删除的物品会在垃圾箱中保留30天，可以恢复。从设置中访问垃圾箱。
