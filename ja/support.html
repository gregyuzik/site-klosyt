お手伝いできることは？

よくある質問への回答は以下をご覧ください。

よくある質問

📸

完璧な写真を撮るコツ

最良の結果を得るには、衣類を無地のコントラストのある背景に掛けてください。明るく均一な照明で、AIが正確に背景を除去し、本来の色を捉えることができます。

📥

インポート方法

写真を撮影、カメラロールからインポート、または衣料品店のURLを貼り付けてアイテムを追加できます。Safariなどのアプリから共有ボタンで直接Klosytに送信することもできます。Klosytは自動的に背景を除去し、クローゼットを美しく保ちます。

👗

コーディネート作成

コーディネートタブでトップス、ボトムス、シューズを組み合わせましょう。レイヤーをドラッグして着こなしの順序で整え、お気に入りのコーディネートを保存できます。

✨

AI提案の使い方

何を着るか迷ったら、AIスタイリストボタンをタップ！デバイス上のAIがリアルタイムの天気とあなたのクローゼットを分析し、あらゆるアクティビティに最適なコーディネートを提案します。

☁️

バックアップと復元

Klosytのデータは大切です。アプリ設定でiCloud同期を有効にして、クローゼット全体を安全にバックアップしましょう。新しいデバイスを入手したら、同じApple IDでサインインすれば、クローゼットがシームレスに復元されます。

📱

マルチプラットフォーム

デジタルクローゼットはどこにでも持ち歩けます。KlosytはiPhone、iPad、Mac、Apple TV、Vision Pro対応のネイティブアプリ — クローゼットはいつでも手元に。

🏷️

アイテムの管理

アイテムをタップして詳細を表示。カテゴリー、シーズン、色、素材、柄を編集できます。詳細画面から直接写真を撮り直すこともできます。削除したアイテムはまずゴミ箱に移動するので復元可能です。

📅

カレンダーと計画

カレンダータブで今後の日やイベントのコーディネートを計画しましょう。保存したコーデを特定の日付に割り当て、着用履歴を追跡し、Appleカレンダーと同期できます。

🔗

共有とエクスポート

お気に入りのコーディネートを美しいパントーンスタイルカードとしてシェアしましょう。iOS共有エクステンションを使って、Safari、Instagram、その他のアプリから直接画像やURLをインポートできます。

🔍

フィルターとソート

KlosytのGelCapフィルターシステムで、クローゼットを瞬時に絞り込めます。カテゴリー、シーズン、色、素材、柄でフィルタリング。名前で検索し、グリッドとクローゼットビューを切り替えられます。

🎨

テーマと壁紙

Klosytを自分好みにカスタマイズ！美しい壁紙やテーマからアプリの見た目をカスタマイズできます — 完全無料、サブスクリプション不要。選んだテーマはすべてのAppleデバイスで同期されます。

📦

Object Capture（3Dスキャン）

LiDAR搭載のiPhone Proモデルで、衣類の素晴らしい3Dスキャンを作成できます。アイテムの周りを歩いて全角度を撮影。3Dモデルは標準写真と一緒に保存されます。

写真を撮影、カメラロールから選択、またはURLを貼り付け。Klosytが自動的に背景を除去します。

2

2. 自動整理

AIがカテゴリー、色、柄などを検出。シーズンタグを追加してAIが適切な時期に提案できるようにしましょう。

3

3. 最初のコーディネートを作成

ライブラリからアイテムを選び、レイヤーの順序をドラッグして調整し、ルックを保存。

4

4. AI提案を取得

アクティビティを選択して、Klosytに天気とクローゼットを分析させ、完璧なコーディネートを提案してもらいましょう。

💡

トラブルシューティング

iCloud同期が動作しない
設定 → Apple ID → iCloudでKlosytのiCloudが有効になっていることを確認。すべてのデバイスで同じApple IDでサインインしていることを確認。Klosytの設定でiCloud同期をオフにしてからオンに戻してみてください。
写真が読み込まない
設定 → Klosyt → 写真でKlosytが写真ライブラリへのアクセス権限を持っていることを確認。画像が空白の場合、iCloud同期を切り替えてみてください。
AI提案が的外れ
アイテムに正確なシーズンタグとカテゴリーがあることを確認。AIはこのメタデータと天気データを使用します。クローゼットのアイテムが多い = 提案が良くなります。
サブスクリプションが認識されない
Klosyt設定に移動し、「購入を復元」をタップ。購入に使用したApple IDでサインインしていることを確認。
背景除去がきれいでない
最良の結果を得るには、均一な照明の下でコントラストのある無地の背景でアイテムを撮影してください。複雑な背景や影を避けてください。
天気や位置情報が更新されない
設定 → プライバシーとセキュリティ → 位置情報サービスでKlosytの位置情報サービスが有効になっていることを確認。天気更新を許可するには「アプリの使用中」または「常に」を選択してください。
アイテムがぼやけて表示される/低画質
画像がまだiCloudからダウンロード中の可能性があります。安定したインターネット接続を確認し、同期が完了するまで待ってください。アプリ内で撮影した写真はフル解像度で保存されています。
アイテムを削除するには？
アイテムをタップ → 詳細画面 → 削除ボタン。削除されたアイテムは30日間ゴミ箱に入り、復元できます。設定からゴミ箱にアクセスできます。
