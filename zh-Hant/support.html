常見問題

📸

拍出完美的照片

為獲得最佳效果，將你的衣物懸掛在純色對比背景前。確保光線明亮均勻，幫助我們的AI準確去除背景並捕捉真實顏色。

📥

如何匯入

透過拍照、從相簿匯入或貼上服裝店URL來加入物品。你還可以從任何應用程式直接分享——在Safari中瀏覽並點擊分享即可傳送到Klosyt。Klosyt會自動去除背景，讓你的衣櫥看起來整潔美觀。

建立搭配

前往搭配標籤頁混搭你的上衣、褲子和鞋子。按穿著順序拖動和整理層次，儲存喜愛的組合以備後用。

✨

使用AI建議

不知道穿什麼？點擊AI造型師按鈕！我們的裝置端智能分析即時天氣和你的真實衣櫥，為每種活動推薦完美搭配。

你的Klosyt資料非常珍貴。在應用程式設定中啟用iCloud同步，安全備份整個衣櫥。取得新裝置後，使用相同的Apple ID登入，你的衣櫥將無縫還原。

多平台

你的數位衣櫥隨你而行。Klosyt是iPhone、iPad、Mac、Apple TV和Vision Pro上的原生應用程式——你的衣櫥始終觸手可及。

行事曆與計劃

使用行事曆標籤為即將到來的日子或活動規劃搭配。將儲存的搭配分配到特定日期，追蹤穿著記錄，並與Apple行事曆同步。

讓Klosyt成為你自己的！從精美的桌布和主題中選擇來自訂應用程式的外觀和風格——完全免費，無需訂閱。你選擇的主題會在所有Apple裝置間同步。

疑難排解

iCloud同步無法運作
確保在設定 → Apple ID → iCloud中為Klosyt啟用了iCloud。確保在所有裝置上使用相同的Apple ID登入。嘗試在Klosyt設定中關閉再開啟iCloud同步。
照片無法載入
檢查Klosyt是否在設定 → Klosyt → 照片中有存取照片圖庫的權限。如果圖片顯示空白，嘗試切換iCloud同步。
AI建議似乎不準確
確保你的物品有準確的季節標籤和類別。AI使用這些中繼資料和天氣資料。衣櫥中物品越多 = 建議越好。
訂閱未被識別
前往Klosyt設定並點擊「恢復購買」。確保使用購買時的Apple ID登入。
背景去除不乾淨
為獲得最佳效果，在純色對比背景上拍攝衣物，確保光線均勻。避免複雜的背景或陰影。
天氣或位置未更新
確保在設定 → 隱私權與安全性 → 定位服務中為Klosyt啟用了定位服務。選擇「使用App期間」或「永遠」以允許天氣更新。
物品顯示模糊或低品質
圖片可能仍在從iCloud下載中。確保網路連線穩定並等待同步完成。應用程式內拍攝的照片以完整解析度儲存。
如何刪除一個物品？
點擊一個物品 → 詳情檢視 → 刪除按鈕。刪除的物品會在垃圾桶中保留30天，可以還原。從設定中存取垃圾桶。
